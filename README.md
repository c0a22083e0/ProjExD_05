# よけろ
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 線と球を避ける。
* ジャンプ可。
* ライフ3つがなくなったらゲームオーバー。
* 参考サイト→https://qiita.com/KMim/items/24a1d69833bc335779a4

## ゲームの実装
###共通基本機能
* 基本行動に関するクラス
* 線とドットの親クラス
* 垂直の線に関するクラス
* 水平の線に関するクラス
* ドットに関するクラス
### 担当追加機能
* プレイヤーの加速
